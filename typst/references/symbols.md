# Symbols Reference

Use `#sym.*` for reliable cross-platform symbols.

## Arrows

```typst
#sym.arrow.r       // → right
#sym.arrow.l       // ← left
#sym.arrow.l.r     // ↔ bidirectional
#sym.arrow.t       // ↑ up
#sym.arrow.b       // ↓ down
```

## Common

```typst
#sym.square        // □ checkbox
#sym.checkmark     // ✓ checkmark
#sym.dot           // · middle dot
#sym.bullet        // • bullet
#sym.dash.em       // — em dash
```

## Math/Logic

```typst
#sym.times         // × multiplication
#sym.div           // ÷ division
#sym.lt.eq         // ≤ less or equal
#sym.gt.eq         // ≥ greater or equal
#sym.approx        // ≈ approximately
#sym.percent       // % percent
```
