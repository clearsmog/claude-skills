# Popular Packages

Install from https://typst.app/universe/ — check for latest versions.

## Drawing & Diagrams

| Package | Purpose | Import |
|---------|---------|--------|
| **cetz** | Core drawing (TikZ-inspired) | `#import "@preview/cetz:0.4.2"` |
| **fletcher** | Flowcharts, automata, arrows | `#import "@preview/fletcher:0.5.9"` |
| **lilaq** | Data visualization, plots | `#import "@preview/lilaq:0.3.0"` |
| **chronos** | Sequence diagrams | `#import "@preview/chronos:0.2.0"` |

## Code & Text

| Package | Purpose | Import |
|---------|---------|--------|
| **codly** | Code blocks with line numbers | `#import "@preview/codly:1.2.0"` |
| **zebraw** | Code listings with annotations | `#import "@preview/zebraw:0.5.0"` |
| **lovelace** | Pseudocode / algorithms | `#import "@preview/lovelace:0.3.0"` |
| **gentle-clues** | Callouts, tips, admonitions | `#import "@preview/gentle-clues:1.0.0"` |

## Presentations & Layout

| Package | Purpose | Import |
|---------|---------|--------|
| **touying** | Presentations (Typst 0.14+) | `#import "@preview/touying:0.6.0"` |
| **tablem** | Markdown-like table syntax | `#import "@preview/tablem:0.2.0"` |
| **showybox** | Customizable text boxes | `#import "@preview/showybox:2.0.4"` |

Note: `polylux:0.3.1` is incompatible with Typst 0.14+ — use `touying` instead.
